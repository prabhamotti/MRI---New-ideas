MRI can be used to verify and map the existing non-ferrous artifacts like porcelain jars. If anybody replaces it with some ordinary jar, then it is very hard to check it. Take a scan for each element. This will yield single 3d figure for each element. In future it will help to found that the jar is replaced. Even after the original got recovered, using MRI scan to scan the recovered jar, and comparing it with the reference, it is possible to conclude that the original one was caught. MRI - Concepts - Prepare the material for scan - Creative Commons License CC-BY-NC-SA

Submitted by prabhamotti on Fri, 09/05/2014 - 05:16

First check for ferromagnetic material inside it. If it is there then clear it out using some chemical procedures.. Then follow the below things.

If the given substance is so dense. Then RF frequency can't penetrate it. If the smashing of the substance allowed then we can powder it,add it to some liquid(water) and make the liquid(water) into gel using some "liquid(water) into gel" powder.

If the given substance is liquid, then make it as gel using some "liquid into gel" powder.
